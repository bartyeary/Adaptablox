# Adaptablox Site

A Next.js application implementing the Adaptablox About page design.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - React components
- `app/globals.css` - Global styles with Tailwind CSS

## Technologies

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Notes

The logo image is loaded from a localhost server at `http://localhost:3845/assets/...`. Make sure the Figma asset server is running if you need to view the logo.
